議事録一発（音声入力修正版）公開手順

【最も確実な使い方】
1. このフォルダ内のファイルをすべてGitHubリポジトリへアップロードします。
2. GitHubの Settings → Pages を開きます。
3. Deploy from a branch、main、/(root) を選び保存します。
4. 表示された https://...github.io/.../ のURLをAndroid版Chromeで開きます。
5. 「音声入力を開始」を押し、マイクを「許可」します。
6. Chromeのメニューから「ホーム画面に追加」するとアプリのように使えます。

【重要】
- ChatGPT内のファイルプレビューではマイクが制限される場合があります。
- 音声認識はChrome等のブラウザサービスを使用するため、音声が認識サービスへ送信される場合があります。
- 議事録、辞書、履歴はブラウザ内に保存されます。
- 自動音声認識が動かない場合は「スマホ標準の音声入力」を押し、Gboardのマイクを使ってください。
