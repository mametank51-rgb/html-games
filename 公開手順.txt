KATO PORTAL 公開更新手順
================================

このZIPは、現在公開中のゲーム6本を残したまま、
実務アプリ4本と新しいトップページを追加する更新セットです。

【トップページの構成】
・実務アプリ
  - 議事録一発
  - かぶ予報ナビ Pro
  - 株トレ｜MARKET JUDGE 500
  - 万象占術｜運命共鳴

・ゲーム
  - 戦国十二州 天下争覇録
  - GRAVITY SHIFT
  - CRASH CORE
  - VOID STRIKER DX
  - 星使いと魔物の王国
  - 月影一閃

【重要】
既存のゲームHTMLはGitHubから削除しないでください。
この更新ZIPには既存ゲーム本体を重複して入れていません。
新しいindex.htmlから、現在GitHubにあるゲームHTMLへリンクします。

【アップロード方法】
1. kato_portal_apps_games_update.zip を解凍します。
2. GitHubの mametank51-rgb/html-games を開きます。
3. Add file → Upload files を選びます。
4. 解凍した中身をすべてアップロードします。
5. index.htmlなどの同名ファイルは上書きします。
6. Commit changes を押します。
7. 数分後、公開ページをChromeで開きます。

公開ページ：
https://mametank51-rgb.github.io/html-games/

【アップロード後の確認】
・最初に「実務アプリ」「ゲーム」の大きな2ボタンが出る
・ゲームを押すと6タイトルが表示される
・実務アプリを押すと4タイトルが表示される
・議事録一発で音声入力開始時にマイク許可が出る

【古い画面が残る場合】
Chromeで公開ページを開き、
右上メニュー → サイトの設定 → データを削除
の後に再読み込みしてください。
